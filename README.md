## Superstore Dataset

## Objective
Build interactive dashboard to track revenue, profit and discout impact

## Steps taken
 - Cleaned and prepared sales data for analysis and visualization
 - Built the required dashboards
 - Identified negative correlation between high discounts and profit margin

## Key Metrics
 - Total Orders
 - Total Profit
 - Total Loss From Discounts
 - Average Profit Per Order

## Additional Metrics
 - Average Discount
 - Total Sales
 - Min Discount
 - Average Discount
 - Max Discount

## Findings
Excessive discount ammounts which lead to orders becoming unprofitable

## Recomendations
Imply regulations on discounts

## Tool Used
Power BI
